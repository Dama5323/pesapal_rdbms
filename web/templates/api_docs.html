Complete reference for PesaPal RDBMS API endpoints

Getting Started

Base URL: http://localhost:8000/api/

All API endpoints require proper authentication. Use the provided API keys or session authentication.

Users API

GET /api/users/

Retrieve all users

POST /api/users/

Create a new user

PUT /api/users/<id>/

Update a user

DELETE /api/users/<id>/

Delete a user

Tasks API

GET /api/tasks/

Retrieve all tasks with JOIN operations

POST /api/tasks/

Create a new task

Test Tasks API

SQL Executor

Execute raw SQL queries against the database
